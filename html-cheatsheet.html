HTML Cheatsheet

This cheatsheet provides a quick reference guide to the basics of HTML.

HTML Document Structure

Here's the basic structure of an HTML document:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <!-- Page content goes here -->
</body>
</html>

HTML Elements

Here are some common HTML elements:

  • <h1> - <h6>: headings
  • <p>: paragraphs
  • <a>: links
  • <img>: images
  • <ul> and <ol>: unordered and ordered lists
  • <li>: list items
  • <div> and <span>: container elements
  • <input>: form input fields

HTML Attributes

Here are some common HTML attributes:

  • id: unique identifier for an element
  • class: group identifier for elements
  • href: URL for a link
  • src: URL for an image
  • alt: alternative text for an image
  • value: value of a form input field
