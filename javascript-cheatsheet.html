JavaScript Cheatsheet

This is a cheatsheet for JavaScript, a popular programming language used for creating dynamic websites and web applications.

Variables

A variable is a container for storing data values. In JavaScript, you can declare a variable using the var, let, or const keyword.

var x = 5;
            let y = "Hello";
const z = true;

Data Types

JavaScript has several built-in data types, including:

  • Numbers
  • Strings
  • Booleans
  • Arrays
  • Objects
  • Null
  • Undefined

Operators

JavaScript has several operators for performing operations on values, including:

  • Arithmetic Operators (+, -, *, /, %)
  • Assignment Operators (=, +=, -=, *=, /=, %=)
  • Comparison Operators (==, !=, ===, !==, <, >, <=, >=)
  • Logical Operators (&&, ||, !)

Conditionals

Conditionals are used to execute code based on a condition. JavaScript has several conditional statements, including:

  • if
  • else
  • else if
  • switch

Loops

Loops are used to repeat code. JavaScript has several loop statements, including:

  • for
  • while
  • do-while
  • forEach

    • Arrays

    An array is a collection of values that can be of any type.

    Creating an array:

    let arr = [1, 2, 3];

    Accessing array elements:

    // Access the first element
            console.log(arr[0]); // Output: 1
            
            // Access the last element
            console.log(arr[arr.length - 1]); // Output: 3

    Iterating over an array:

    for (let i = 0; i < arr.length; i++) {
              console.log(arr[i]);
            }

    Objects

    An object is a collection of key-value pairs.

    Creating an object:

    let person = {
              name: 'John Doe',
              age: 30,
              email: 'john.doe@example.com'
            };

    Accessing object properties:

    // Dot notation
            console.log(person.name); // Output: John Doe
            
            // Bracket notation
            console.log(person['age']); // Output: 30

    Iterating over an object:

    for (let key in person) {
              console.log(key + ': ' + person[key]);
            }

    Functions

    A function is a block of code that can be called to perform a specific task.

    Creating a function:

    function add(a, b) {
              return a + b;
            }

    Calling a function:

    console.log(add(2, 3)); // Output: 5

    Function expressions:

    const multiply = function(a, b) {
              return a * b;
            };
            
            console.log(multiply(2, 3)); // Output: 6

    Arrow functions:

    const divide = (a, b) => a / b;
            
            console.log(divide(6, 3)); // Output: 2
