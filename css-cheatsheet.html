CSS Selectors

Selector Description
* {} Selects all elements
#id {} Selects an element with a specific id
.class {} Selects elements with a specific class
element {} Selects all elements of a specific type
element, element {} Selects all elements of the first type and all elements of the second type
element.class {} Selects all elements of a specific type with a specific class
element > element {} Selects all direct child elements of the first element that are of the second type
element + element {} Selects the first element that is immediately after the second element
element ~ element {} Selects all elements that are siblings of the second element and come after it

CSS Box Model

  • Content - The content of the box, where text and images appear
  • Padding - Clears an area around the content
  • Border - A border that goes around the padding and content
  • Margin - Clears an area outside the border

CSS Grid

CSS Grid is a powerful layout system that allows you to create complex grids for your website.

Creating a Grid Container

To create a grid, you first need to define a grid container. This is done by setting the display property to grid or inline-grid on an element.


          .grid-container {
            display: grid;
            grid-template-columns: 1fr 1fr 1fr;
            grid-template-rows: 100px 100px;
            grid-gap: 20px;
          }

This creates a grid with 3 columns and 2 rows, with a 20px gap between each cell.

Creating Grid Items

Once you have a grid container, you can start adding grid items to it. These are the elements that will be placed inside the grid cells.


          .grid-item {
            grid-column: 2 / 3;
            grid-row: 1 / 3;
          }

This code sets the grid item to span from the second to the third column, and from the first to the third row.

Grid Properties

There are many properties you can use to customize your CSS Grid, such as grid-template-columns, grid-template-rows, grid-template-areas, and more. For a full list of properties and their values, check out the Mozilla Developer Network.
